# Daily Korean Political News

매일 한국 정치/사회 뉴스를 자동 수집하고, 보수·진보 양쪽 시각으로 요약합니다.

**https://zer0-kr.github.io/daily-news/**

## 어떤 사이트인가요?

매일 오전 7시(KST)에 전날의 주요 정치/사회 뉴스를 자동으로 수집합니다. 9개 매체(조선일보, 동아일보, 한국경제, 경향신문, 한겨레, 오마이뉴스, SBS, 연합뉴스, 뉴시스)의 기사를 클러스터링하여 상위 5개 스토리를 선정하고, 각 스토리에 대해 보수와 진보 양쪽 시각의 분석을 제공합니다.

## 뉴스 구조

각 날짜의 뉴스는 다음과 같이 구성됩니다:

- **헤드라인 개요** — 오늘의 주요 뉴스 5개, 매체 수 및 진영별 보도 비중
- **배경** — 해당 이슈의 맥락 설명
- **보수 시각** — 보수 진영의 관점과 논리
- **진보 시각** — 진보 진영의 관점과 논리
- **출처** — 원본 기사 링크 (매체별)

## 기술 스택

| 구성 요소 | 기술 |
|-----------|------|
| 뉴스 수집 | RSS 피드 + NewsData.io API |
| 클러스터링 | TF-IDF 기반 유사도 분석 |
| 요약 | Claude API |
| 사이트 | Jekyll + GitHub Pages |
| 자동화 | GitHub Actions (매일 오전 7시 KST) |

## 로컬 실행

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
```
